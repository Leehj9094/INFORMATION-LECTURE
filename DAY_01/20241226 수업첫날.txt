1. Network 는 무엇이며 무엇을 목적으로 하는가?

+데이터교환을 목적으로 여러 통신장치들을 연결한 통신망
2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
LAN / WAN 
3. LAN 의 특징에 대해서 서술하세요
단거리 적으로 가능하며 우리가 일상 생활 회사나 학교 등에서 쓰는 정도의 연결 장치, 개인 사용
+ 근거리 통신망 , 연결된 장치간의 거리가 비교적 가까운 형태로 연결한 통신
4. WAN의 특징에 대해서 서술하세요
원거리로 LAN과는 다르게 광범위한 사용으로 다른 지역은 물론 전세계 어디서나 가능한, 기업 (통신사)
+원거리통신망(광역통신망), 통신장치간 거리가 비교적 먼 거리의 통신장치간의 연결형태를 가진 통신망
5. InterNet 이란
거대 통신망
+ 여러 종류의 통신망을 한데 묶은 거대 통신망
6. 프로토콜은 무엇인가?
사전에 약속(저장)된 데이터를 나타내는 것
+ 통신을 하기 위해 사전에 정의(약속)한 규칙/규정
7. 네트워킹 모델의 목적은 무엇인가?
관심, 목표, 가치 또는 경험을 공유하는 사람들과 관계를 구축하고 유지하는 프로세스
+ 통신망 구현을 위한 지침서, 하자 발생시 참고하는 지침서, 이기종간 통신을 위해 미리 정의해 놓은 표준 프로토콜 모델
8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
TCP/IP 
- 4계층 구조로 OSI의 1,2,5,6,7 계층을 통합한 4계층으로 구성
OSI 7
- 1~7까지의 계층으로 나뉘어 있으며 
1층은 물리적인 선 등 + 물리: 장치간 물리적인 연결
2층 데이터 기준점으로 프레임을 구성, + 데이터링크: 장치간 데이터 운송방식 결정
3층 네비게이션과 같은 최적의 경로 탐색 이나 전송을 함, + 네트워크: 경로탐색
4층 전송 단계로서 송수신지의 역할의 담당하며 오류제어, 재전송 등을 담당 + 전송 : 송수신간의 데이터의 신뢰성을 확보
5층 구간 연결 관리 및 통신 오류 복구 및 재전송, 
6층 데이터를 상호 인식 가능하도록 변환 (인코딩, 암호화, 압출, 코드 변환 등)
7층 우리가 사용하고 있는 앱, 메일, 메세지, OTT 등을 사용하는 

1 물리계층 하는일/역할/관련 장치는?
장치 간 장치 연결 / 신호 연결 하기 위한 / 케이블 선, 허브, 리피터
+ 역할 : 장치의 물리적 연결 / 약속(프로토콜) : 전기적 신호전달(PUD:Bit) / 장치 : 케이블, RJ45, 허브, 리피터 ,,
2 Cable 이란
네트워크 장비나 장치를 연결하는 데 사용되는 물리적인 매체 , 전기를 이용해서 신호를 전달하는 전기선
+ 전선 전기를 이용하여 신호를 전달하는 선
3 LAN Cable 종류
이더넷 케이블, 광섬유 케이블, 동축 케이블
+ STP / UTP : 많이 사용 / FTP
4 부호(Code) 란?
약속한 신호를 가지고 있는 기호
+ 부호의 전달형태
5 신호(Signal)이란?
원격지로의 부호 전달
+ 부호의 전달
6 신호전달에 필요한 요소?
전기에너지와 통신망
+ 통신망, 통신기기, 에너지원
7 리피터란?

+ 신호증폭기
8 허브란

+ 접선장치
