다이렉트 게임 구매 사이트

판매자
- 판매자는 사이트에 판매할 게임을 올릴 수 있다
- 판매자는 사이트에 올린 게임을 수정할 수 있다
- 판매자는 사이트에 올린 게임을 삭제할 수 있다
- 판매자는 사이트에 이벤트를 할 수 있다?
- 판매자는 사이트에서 회원가입을 할 수 있다?
- 판매자는 

구매자 
- 구매자는 사이트에 올라온 게임을 살 수 있다
- 구매자는 사이트에 올라온 게임을 구경할 수 있다
- 구매자는 사이트에서 반품을 할 수 있다
- 구매자는 사이트에서 회원가입을 할 수 있다
- 구매자는 사이트 내에 질문을? 의견? 댓글?을 달 수 있다?
- 구매자는 사이트에서 진행하는 이벤트에 참여 할 수 있다?



회원 
- 회원은 로그인 할 수 있다
- 회원은 로그아웃 할 수 있다
- 회원은 개인정보를 수정 할 수 있다
- 회원은 개인정보를 조회 할 수 있다
- 회원은 판매자가 올린 상품을 구매할 수 있다



비회원 
- 비회원은 사이트 내에 검색을 할 수 있다
- 비회원은 사이트 내 구경할 수 잇다?

관리자
- 관리자는 등록된 상품을 수정할 수 있다
- 관리자는 등록된 상품을 삭제 할 수 있다
- 관리자는 판매자를 대신하여 상품을 등록/조정 할 수 있다
- 관리자는 판매자를 대신하여 상품의 이벤트를 등록/조정 할 수 있다
- 관리자는 등록된 상품들을 조회할 수 있다 
- 관리자는 구매자, 판매자 등의 게시판을 조정할 수 있다?
