보조 입출력 스트림
- byte 단위로 가지고 와서 인풋 아웃풋으로 바꾸는(기본 스트림을 인자로 전달하여 사용됨)
- 단독으로 사용 불가(항상 기본이랑 같이 사용되야 함)
- output으로 올린 순서대로  input을 들고 와야 순서가 섞이지 않음

직렬화(serialize)
: java 에서 사용되는 object나 data를 다른 컴퓨터의 시스템에서도 사용 할 수 있도록 byteStream 형태로 연속적인 데이터로 변환하는 포맷 변환 기술
- 자바의 고유 기술로 당연히 자바 안에서 최적화 되어 있음
- 광활할 레퍼런스 타입에 대해 제약 없이 외부로 내보낼 수 있음

역직렬화(Deserialize)
: byte로 변환된 data를 원래대로 시스템에 object 또는 Data로 변환하는 기술

byteStream
: 서버 간에 출발지 목적지로 입출력하기 위한 데이터가 흐르는 통로
